# Nimで自作Shellを書いてみよう
> 参考：
> - https://brennan.io/2015/01/16/write-a-shell-in-c/
> - https://github.com/brenns10/lsh

## nimlsh
- Type program names and argumetns, and hit enter.
- The following are built in:
    - cd
    - pwd
    - ls
    - exit
    - help
