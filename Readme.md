# nimlsh: A small shell written in Nim

## Prerequisites
- Having [nim](https://nim-lang.org/) installed.

## How to run
- Open the folder and type ```nim c -r nimlsh.nim``` in the terminal.

## Features
- The following are built in:

  ```
  $ cd
  $ ls
  $ pwd
  $ exit
  $ help 
  ```
